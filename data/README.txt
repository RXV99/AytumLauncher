Java ME MIDlet Storage
=======================
Copy your .jar and .jad files into this directory on your PS Vita:

    ux0:data/java/

The launcher will automatically detect all .jar files and parse
any .jad files found alongside them for app metadata.

Supported file types:
  - *.jar   (Java ARchive containing MIDlet classes)
  - *.jad   (Java Application Descriptor with metadata)

Place them here on your Vita (via USB/FTB):
  ux0:data/java/MyGame.jar
  ux0:data/java/MyGame.jad   (optional, for metadata)
